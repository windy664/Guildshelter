# GuildShelter 公会营地

GuildShelter 是一套给 Minecraft 服务器使用的“公会营地 / 庄园地皮”系统。

它的目标很简单：让每个公会都有自己的营地世界，让成员能拥有自己的庄园地皮，同时把参观、权限、升级、互动、管理这些日常操作整理成一套清楚的玩法。

## 功能亮点

### 每个公会一个专属营地

公会可以拥有独立的营地世界。成员进入后可以建设主城、分配庄园、设置传送点，让公会不再只是聊天频道，而是一个真正可参观、可建设、可经营的家。

### 成员庄园地皮

成员可以领取自己的庄园地皮，用来建房、布置家具、存放物资或展示作品。

庄园有等级和可建设范围，升级后可以逐步扩大可用空间，而不是一开始就把所有区域全部开放。

### 庄园控制器 GUI

玩家可以通过庄园控制器打开一个统一菜单，查看和管理常用功能：

- 庄园资料
- 庄园升级
- 回家和设置 Home
- 成员与权限入口
- 访客开放与关闭
- 留言、评分、送花等互动功能

玩家不需要记一大堆命令，常用操作可以从菜单进入。

### 庄园升级消耗

庄园升级可以配置金币和物品消耗。

升级时会先检查玩家是否满足全部条件，确认金币和材料都够以后才会扣除，避免出现扣了钱但材料不够的情况。

### 访客参观

庄园可以临时开放给访客参观，也可以关闭访问。

适合公会活动、建筑展示、庄园评选、好友参观等场景。

### 地图高亮支持

进入公会营地后，玩家可以在地图上看到对应区域高亮，方便分辨庄园、道路和公会范围。

这对大型营地尤其有用，玩家不容易迷路，也更容易找到自己的地皮。

### 权限与保护

系统会保护庄园、主城和道路区域，避免无权限玩家随意破坏。

庄主可以管理自己的庄园成员、黑名单和访问规则；公会管理者可以管理主城区域。

### 道路按需生成

公会营地的道路不会一次性全部铺完。

玩家走到哪里，系统再按需补上附近道路，减少创建庄园或扩展营地时的卡顿。

### 留言、评分与互动

玩家可以给庄园留言、评分、送花，让建筑展示和公会活动更有参与感。

适合做庄园评选、参观活动、优秀建筑展示等玩法。

### 管理员工具

管理员可以创建公会营地、传送到公会世界、分配庄园、升级公会、查看世界列表、卸载营地等。

删除公会营地时，系统会清理插件内的核心数据，并提示对应的世界存档文件夹，方便管理员在后台确认后手动删除，避免误删重要存档。

## 适合什么服务器

- 有公会、营地、家园、庄园玩法的服务器
- 想给每个公会独立建设空间的服务器
- 想做庄园展示、建筑评选、玩家参观活动的服务器
- 想要更清楚的地皮保护和成员权限管理的服务器

## 玩家常用入口

打开庄园控制器：

```text
/gs controller
```

也可以使用：

```text
/gs gui
```

回到自己的庄园：

```text
/gs home
```

升级庄园：

```text
/gs upgrade
```
